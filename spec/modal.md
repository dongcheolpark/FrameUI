# FrameUI Component Spec

## 1. 컴포넌트 개요

- 컴포넌트 이름: Modal
- 한 줄 설명: 사용자 인터랙션을 차단하고 중요한 정보를 화면 중앙에 노출하는 레이어 컴포넌트

## 2. 해결할 문제

- 페이지 전환 없이 사용자에게 추가 정보나 확인 절차를 제공해야 한다.
- 모달이 열렸을 때 배경 콘텐츠와의 시각적 분리(Overlay)가 필요하다.
- Controlled(isOpen)와 Uncontrolled(defaultOpen) 모드를 모두 지원하여 유연성을 확보한다.

## 3. 필요한 필수 기능

1. **상태 제어**: 모달의 열림/닫힘 상태를 토글할 수 있어야 한다.
2. **이벤트 핸들링**: 닫기 버튼, 오버레이 클릭 시 `onOpenChange` 콜백을 호출해야 한다.
3. **접근성**: `role="dialog"` 및 `aria-modal="true"` 속성을 지원해야 한다.
4. **스타일링**: 상태에 따른 애니메이션 처리를 위해 `data-state` ("open" | "closed")를 노출해야 한다.
