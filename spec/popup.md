# FrameUI Component Spec

## 1. 컴포넌트 개요

- 컴포넌트 이름: Popup (Toast)
- 한 줄 설명: 화면 구석에 일시적으로 나타나 사용자에게 피드백을 제공하는 알림 컴포넌트

## 2. 해결할 문제

- 작업 흐름을 방해하지 않으면서 성공, 실패, 정보 등의 메시지를 전달해야 한다.
- 일정 시간이 지나면 자동으로 사라지는 기능(Auto-dismiss)이 필요하다.
- 화면의 다른 요소들보다 항상 위에 노출되어야 한다.

## 3. 필요한 필수 기능

1. **자동 소멸**: `duration` 프로퍼티를 통해 설정된 시간 후 자동으로 닫혀야 한다.
2. **유형 지원**: `type` ("info" | "success" | "error")에 따라 시각적 테마가 변경되어야 한다.
3. **수동 제어**: 사용자가 직접 닫기 버튼을 눌러 제거할 수 있어야 한다.
4. **시각적 효과**: 등장과 퇴장 시 부드러운 전환을 위해 `data-state` 속성을 제공해야 한다.
