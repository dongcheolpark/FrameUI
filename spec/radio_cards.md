# FrameUI Component Spec

## 1. 컴포넌트 개요

- 컴포넌트 이름: Radio Cards
- 한 줄 설명: 여러 옵션 중 하나만 선택할 수 있도록 카드 형태로 제공되는 접근성 기반 단일 선택 컴포넌트이다.

## 2. 해결할 문제

- 기본 radio input은 기능적으로 충분하지만, 각 옵션에 설명, 아이콘, 보조 정보 등 더 풍부한 내용을 함께 보여주기에는 표현력이 제한적이다.
- 사용자가 여러 선택지를 한눈에 비교하면서도 하나만 선택해야 하는 상황에서, 일반적인 라디오 버튼보다 더 직관적인 카드형 UI가 필요하다.
- 키보드 조작과 스크린리더 환경에서도 일관된 단일 선택 상호작용을 제공해야 한다.
- Controlled/Uncontrolled 모두 지원해서 다양한 폼 환경에서 유연하게 사용하고 싶다.
- Checkbox Cards와 유사한 사용성을 유지하면서도, 단일 선택이라는 차이에 맞는 명확한 API와 동작이 필요하다.

## 3. 필요한 필수 기능

1. 여러 카드 중 하나의 옵션만 선택할 수 있어야 한다.
2. 카드 전체를 클릭했을 때 해당 옵션이 선택되어야 한다.
3. `value`, `defaultValue`, `onValueChange` API를 제공해야 한다.
4. 비활성화(disabled) 상태를 지원해야 한다.
5. 키보드 탐색과 선택이 가능해야 한다.
6. 스크린리더가 인식할 수 있도록 `radiogroup`, `radio`, `aria-checked` 등 적절한 접근성 속성을 제공해야 한다.
7. 스타일링과 상태 표현을 위해 선택 여부 및 비활성화 상태를 외부에서 식별 가능하게 노출해야 한다.
