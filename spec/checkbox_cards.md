# FrameUI Component Spec

## 1. 컴포넌트 개요

* 컴포넌트 이름: Checkbox Cards
* 한 줄 설명: 여러 옵션을 카드 형태로 표시하고, 각 카드를 체크박스처럼 복수 선택할 수 있는 접근성 기반 선택 컴포넌트

---

## 2. 해결할 문제

* 기본 체크박스 리스트는 옵션의 제목, 설명 등 추가 정보를 함께 보여주기에 시각적으로 한계가 있다.
* 카드 형태의 선택 UI를 제공하면서도 체크박스와 동일한 의미론적 구조와 접근성을 유지하고 싶다.
* 카드 전체 영역을 클릭 가능하게 만들어 사용자 경험을 개선하고 싶다.
* 복수 선택이 가능한 옵션 그룹을 직관적인 형태로 표현하고 싶다.
* Controlled/Uncontrolled 모두 지원해서 폼, 필터, 설정 화면 등 다양한 상황에서 유연하게 사용하고 싶다.

---

## 3. 필요한 필수 기능

1. 각 카드를 클릭 또는 키보드 입력으로 선택/해제할 수 있어야 한다.
2. `value`, `defaultValue`, `onValueChange` API를 제공해야 한다.
3. 비활성화(`disabled`) 상태를 지원하고, 스타일링을 위해 상태(`data-state`)를 노출해야 한다.
